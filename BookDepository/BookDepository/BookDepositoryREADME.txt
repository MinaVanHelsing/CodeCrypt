=======
Book Depository
=======
BookSearch is an ASP.NET MVC application 
that allows users to enter books into a 
local database and execute a simple search
by ID, author, title, genre or price.

============
Installation
============

Visual Studio must be installed before
running this application. In Visual Studio,
please select IIS Express when running the 
code.

============
The local database
============
If the local base is empty, users can
enter their own books by selecting the 
Create option on the Books Index page.
